# koas-cli
A simple CLI for creating a koa project

# Installation
Prerequisites: [Koa](https://github.com/koajs/koa) v2.x, [Node.js](https://nodejs.org/en/) (>=7.6.0), npm version 3+.

```
$ npm install koas-cli -g
```

# Usage
Open your terminal and type:

```
$ koas <project-name>
```

Example:

```
$ koas hello-koa
```
